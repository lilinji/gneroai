# CHESS超算门户简介

CHESS（计算增强科学服务中心）门户是太行I号超算平台访问和管理资源的主要Web界面。这个综合性门户为新手和经验丰富的用户提供直观的基于浏览器的高性能计算任务环境。

## 平台访问

### 登录流程

通过Web浏览器访问CHESS门户：**https://webhpc.tibhpc.net/**

<img src="/static/guides/hpc/platform-overview/chess-portal/login-page.png" alt="CHESS门户登录页面" width="800" />

输入您的用户名和密码凭据登录到系统桌面环境。

### 初始设置

成功登录后，您将看到主仪表板界面：

<img src="/static/guides/hpc/platform-overview/chess-portal/dashboard-overview.png" alt="CHESS门户仪表板" width="800" />

## 主要能力

### Web界面
- **浏览器访问**：无需安装软件
- **跨平台**：兼容Windows、macOS和Linux
- **响应式设计**：针对桌面和移动设备优化
- **安全通信**：所有数据传输采用HTTPS加密

### 集成文件管理
- **Web文件浏览器**：完整的文件系统导航
- **文件操作**：上传、下载、复制、移动、删除
- **权限管理**：配置文件和目录访问权限
- **实时同步**：跨会话即时更新

### 作业编排
- **可视化作业提交**：GUI驱动的作业配置
- **队列管理**：监控和管理作业队列
- **资源分配**：指定计算资源和需求
- **执行监控**：实时作业状态和进度跟踪

### 系统监控
- **资源利用率**：跟踪CPU、内存和存储使用情况
- **性能指标**：监控集群性能指标
- **使用分析**：生成详细使用报告
- **成本管理**：跟踪计算成本和计费

## 用户工作流程

### 入门指南
1. **账户设置**：从系统管理员处获取凭据
2. **初次登录**：访问门户并完成个人资料配置
3. **环境熟悉**：探索仪表板和可用功能
4. **资源检查**：查看可用队列、存储和限额

### 基本操作
1. **文件管理**：上传数据文件并组织目录结构
2. **应用安装**：从应用中心下载所需软件
3. **作业准备**：创建作业脚本并配置执行参数
4. **资源监控**：跟踪使用情况并管理计算成本

### 高级用法
1. **工作流自动化**：设置定时任务和自动化流程
2. **容器部署**：为专门工作负载管理自定义容器镜像
3. **性能优化**：监控和优化资源利用率
4. **协作工作**：与团队成员共享资源和结果

## 支持与培训
