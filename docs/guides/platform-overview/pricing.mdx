## 🎯 收费概述

### 基本原则

智算平台遵循"资源共享、开放合作、服务创新、安全高效"的基本原则，采用后付费方式，根据统一的资源调度，用户使用相应的资源。

### 收费模式

- **排队模式**：按实际使用时间计费
- **独占模式**：相比排队模式费用上调20%
- **所内用户**：按照标准费率
- **所外用户**：按照1.5倍收费标准倍率

### 免费资源

平台提供部分免费资源供用户试用：
- 免费试用系统（虚拟化CPU/GPU节点）
- 每用户50GB免费家目录
- 收费账户每用户免费配置1TB家目录

## 💾 存储资源收费标准

### 存储配置

| 存储类型 | 免费配额 | 收费标准 | 挂载点 | 说明 |
|---------|---------|---------|--------|------|
| **家目录** | 1TB/用户 | 40.00元/1TB月 | `~` | 个人工作目录 |
| **备份目录** | 无 | 30.00元/1TB月 | `~/backup` | 数据备份存储 |
| **在线备份** | 无 | 30.00元/1TB月 | - | 在线备份服务 |
| **离线备份** | 无 | 450元/1TB年 | - | 磁带库备份 |

### 计费规则

:::warning 重要提醒
存储账户计费规则：

1. **每日统计**：每天固定时间点统计用户 `fhome`、`fproject`、`fbackup` 目录中的**当日存储用量**作为该用户当日的存储用量

2. **计费周期**：在计费周期（通常为自然月）内统计用户存储用量的**最大值（以TB为单位，不足1TB按1TB算）**作为该计费周期内该账户的**计费周期存储用量**

3. **计费公式**：计费单元在计费周期内存储收费金额为：
   ```
   计费金额 = fhome计费周期存储用量 × 0 + (fproject计费周期存储用量 - 1TB) × 40 + fbackup计费周期存储用量 × 30
   ```
:::

## 🏢 用户托管资源收费标准

### 设备托管费用

| 托管类型 | 年收费标准 | 说明 |
|---------|----------|------|
| **整机柜** | 24万元 | 单机柜26KW，42U |
| **普通机架式服务器** | 1万元/2U | 标准机架服务器 |
| **高密度服务器** | 按24万元×(被占用空间/42U)和24万元×(实际功耗/26KW)较高者 | 或酌情处理 |
| **配套设备** | 0.5万/U | 交换机、管理登录存储节点等 |

### 特殊收费规定

:::caution 超量超期收费
- **超量收费**：每用户（以课题组为单位）设备托管超过两个机柜的，超出部分按照基本收费标准的1.5倍收取托管费
- **超期收费**：托管时间超过5年的用户设备托管时，超出5年的时间，按照前期收费标准的2倍收取托管费
:::

## 📊 计费规则

### 计费方式

- **后付费**：根据统一的资源调度，用户使用相应的资源
- **计费周期**：通常为自然月
- **计费精度**：按小时计费，不足1小时按1小时计算
- **计费单位**：CPU按核时，GPU按卡时，存储按TB月

### 费用管理

- **所内用户**：原则上使用测试加工费支出智算平台服务费用
- **费用用途**：主要用于设备维修、耗材配件购买、培训组织和交流活动等费用
- **支付方式**：支持银行转账、支付宝、微信支付等多种方式

### 资源调度

:::warning 重要提醒
在有重大应用项目需要时，智算平台可能会临时独占节点或暂停收费队列以保障重大应用项目需要。但会及时通知用户，并预留充分的时间供用户操作。
:::

## 👥 用户权利与义务

### 用户权利

1. **资源申请**：用户在使用智算平台环境过程中，可以根据需求申请资源变更
2. **资源使用**：在遵守服务协议的前提下，合理使用平台资源
3. **技术支持**：获得平台提供的技术支持和培训服务
4. **信息透明**：了解资源使用情况和费用明细

### 用户义务

1. **遵守协议**：遵守服务协议，不利用计算资源从事与其申请计算内容无关的计算活动（包括且不限于挖矿），不恶意耗费计算资源与网络流量，不进行可能影响高性能环境的操作

2. **按时缴费**：依照服务协议按时缴纳费用。对于有意拖欠或拒交费用的用户，智算平台有权停止其使用权限。由此造成的损失由用户负责

3. **账号安全**：维护自身的账号与密码安全。由于自身泄漏引起的账号盗用发生的一切损失，由用户承担

4. **信息更新**：用户信息发生变动时，及时向智算平台提出信息变更申请

5. **成果标注**：用户有义务对利用本平台产生的科研成果进行标注

6. **数据备份**：除特殊与智算平台约定由平台备份的情况外，用户应定期对数据进行备份。用户存储在平台上的数据由用户负责保管，本平台不承担用户在集群中数据丢失的责任
