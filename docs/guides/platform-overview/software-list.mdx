## 🎯 软件概述

### 软件分类

智算平台为用户提供多种类型的软件工具，满足不同使用场景的需求：

- **终端登录类**：用于远程连接和管理服务器
- **VPN接入类**：用于安全访问平台资源
- **文件传输类**：用于数据传输和文件管理
- **虚拟化类**：用于连接虚拟桌面环境
- **编程工具类**：用于代码开发和调试

### 支持平台

所有软件均支持主流操作系统：
- **Windows**：Windows 10/11
- **macOS**：macOS 10.15+
- **Linux**：Ubuntu 18.04+, CentOS 7+, 麒麟/UOS等

## 💻 终端登录类

### Tabby（推荐）

:::tip 跨平台终端工具
**Tabby** 是一款现代化的、跨平台的终端模拟器，支持 **SSH、SFTP、Mosh** 等协议。它的目标是提供一个简洁、高效并且具有良好用户体验的终端工具。

**特点**：
- 支持 Windows、Linux 和 macOS
- 现代化界面设计
- 多标签页支持
- 内置SFTP文件管理器
- 主题和插件系统
:::

| 平台 | 下载地址 | 版本 | 文件大小 |
|------|---------|------|---------|
| Windows | [GitHub Releases](https://github.com/Eugeny/tabby/releases/tag/v1.0.222) | v1.0.222 | ~50MB |
| Linux | [GitHub Releases](https://github.com/Eugeny/tabby/releases/tag/v1.0.222) | v1.0.222 | ~50MB |
| macOS | [GitHub Releases](https://github.com/Eugeny/tabby/releases/tag/v1.0.222) | v1.0.222 | ~50MB |

### Xshell

**Xshell** 是一款功能强大的Windows平台下的SSH、Telnet和Rlogin客户端，广泛用于远程访问和管理Unix/Linux服务器。

**特点**：
- 专业的SSH客户端
- 多会话管理
- 脚本自动化
- 安全连接

| 平台 | 下载地址 | 版本 | 说明 |
|------|---------|------|------|
| Windows | [官方网站](https://www.xshell.com/zh/) | 8.0.0055p | 需要注册账号 |

### MobaXterm

**MobaXterm** 结合了SSH客户端和多种Unix工具的Windows应用，提供增强的功能，如图形化文件传输。

**特点**：
- 集成SSH客户端
- 内置SFTP文件管理器
- Unix工具集成
- 多标签页界面

| 平台 | 下载地址 | 版本 | 说明 |
|------|---------|------|------|
| Windows | [官方网站](https://mobaxterm.mobatek.net/) | v24.2 | 免费版和专业版 |

### PuTTY

**PuTTY** 是一款流行的Windows SSH客户端，具有简单的图形用户界面，支持SSH、Telnet等协议。

**特点**：
- 轻量级设计
- 开源免费
- 简单易用
- 广泛支持

| 平台 | 下载地址 | 版本 | 说明 |
|------|---------|------|------|
| Windows | [官方网站](https://www.putty.org/) | 0.81 | 开源免费 |

### SecureCRT

**SecureCRT** 是专业的终端仿真程序，支持SSH和其他多种协议，常用于企业环境。

**特点**：
- 企业级功能
- 高级会话管理
- 脚本支持
- 安全连接

| 平台 | 下载地址 | 版本 | 说明 |
|------|---------|------|------|
| Windows | [官方网站](https://www.vandyke.com/products/securecrt/) | 9.5.2.3325 | 商业软件 |

## 📖 使用指南

### 软件选择建议

#### 终端登录工具选择

:::tip 推荐选择
- **Windows用户**：推荐使用Tabby或MobaXterm
- **Linux/macOS用户**：推荐使用Tabby
- **企业用户**：推荐使用SecureCRT或Xshell
:::

#### VPN客户端选择

- **Windows用户**：使用aTrust Windows客户端
- **Ubuntu用户**：使用aTrust Ubuntu客户端
- **macOS用户**：使用aTrust macOS客户端
- **国产系统用户**：根据架构选择对应版本

#### 文件传输工具选择

- **Windows用户**：推荐使用WinSCP或MobaXterm
- **需要图形化界面**：选择WinSCP或MobaXterm
- **命令行用户**：使用scp或rsync命令

### 安装和配置

#### 通用安装步骤

1. **下载软件**：从官方或本地下载链接获取安装包
2. **安装软件**：按照安装向导完成安装
3. **配置连接**：输入服务器地址、用户名和密码
4. **测试连接**：验证连接是否正常

#### 常见问题解决

:::warning 常见问题
1. **连接失败**：检查网络连接和服务器地址
2. **认证失败**：确认用户名和密码正确
3. **权限问题**：联系管理员获取访问权限
4. **软件兼容性**：确保使用最新版本
:::

### 安全使用建议

1. **定期更新**：保持软件版本最新
2. **安全连接**：使用SSH等加密协议
3. **密码管理**：使用强密码并定期更换
4. **权限控制**：只授予必要的访问权限
