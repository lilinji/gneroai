# 平台概览文档静态资源

## 目录说明

本目录用于存放平台概览相关文档的静态资源，包括图片、附件、文档等。

## 文件组织

- `attachments/` - 存放文档相关的附件文件
- `documents/` - 存放官方文档和表格模板
- `images/` - 存放文档中使用的图片
- `diagrams/` - 存放架构图和流程图

## 注意事项

1. 所有静态资源应保持合理的文件大小
2. 文档格式优先使用PDF或DOC格式
3. 附件文件应提供清晰的命名和说明
4. 定期清理不再使用的资源文件
5. 确保所有链接路径正确无误
