# 软件列表文档静态资源

## 目录说明

本目录用于存放软件列表相关文档的静态资源，包括软件安装包、图片、文档等。

## 文件组织

- `terminal/` - 存放终端登录工具安装包
- `vpn/` - 存放VPN客户端安装包
- `file-transfer/` - 存放文件传输工具安装包
- `virtualization/` - 存放虚拟化工具安装包
- `development/` - 存放编程工具安装包
- `images/` - 存放文档中使用的图片
- `documents/` - 存放相关文档

## 软件分类

### 终端登录工具
- Tabby - 跨平台终端模拟器
- Xshell - Windows SSH客户端
- MobaXterm - 多功能终端工具
- PuTTY - 轻量级SSH客户端
- SecureCRT - 专业终端工具

### VPN客户端
- aTrust Windows客户端
- aTrust Ubuntu客户端
- aTrust macOS客户端
- 麒麟/UOS客户端（多种架构）

### 文件传输工具
- WinSCP - Windows文件传输工具
- Xftp - Windows FTP客户端
- MobaXterm - 集成文件传输功能

### 虚拟化工具
- VMware Horizon Client - 虚拟桌面客户端

### 编程工具
- Visual Studio Code - 轻量级代码编辑器
- PyCharm - Python IDE
